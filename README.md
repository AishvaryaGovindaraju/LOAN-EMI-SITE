# EMI Calculator India 

## Folder Structure
```
emi-site/
├── index.html                 ← Main EMI Calculator (homepage)
├── style.css                  ← Shared styles for ALL pages
├── script.js                  ← Main calculator logic
├── privacy.html               ← Required for AdSense
├── about.html                 ← Required for AdSense
├── contact.html               ← Required for AdSense
├── robots.txt                 ← Tells Google to index your site
├── sitemap.xml                ← Helps Google find all pages
├── vercel.json                ← Deployment config
│
└── pages/
    ├── home-loan.html         ← Home Loan EMI Calculator
    ├── personal-loan.html     ← Personal Loan EMI Calculator
    ├── car-loan.html          ← Car Loan EMI Calculator
    ├── education-loan.html    ← Education Loan EMI Calculator
    ├── bank-compare.html      ← Bank Rate Comparison (killer feature)
    ├── prepayment.html        ← Prepayment Savings Simulator
    └── affordability.html     ← Salary → Max Loan Checker
```

## Test Locally

1. Download/unzip all files into a folder called `LOAN & EMI CACL`
2. Double-click `index.html` → opens in browser
3. Test all features: EMI calc, sliders, comparison, amortization table
4. Done. No server needed for HTML/CSS/JS.
